Acerca de IES Cavanilles

El Instituto Cavanilles de Alicante es una institución académica dedicada a la investigación y formación en el campo de la biología y la conservación ambiental. Fue fundado en 1989 y lleva el nombre del botánico español Antonio José Cavanilles. El instituto cuenta con un equipo de investigadores y profesores altamente cualificados que trabajan en proyectos relacionados con la biodiversidad, la ecología, la evolución y la conservación de especies y ecosistemas. Además, ofrece programas de posgrado y cursos de formación para estudiantes y profesionales interesados en estos campos.

Nuestros mejores Cursos y Programas

Curso de Front-End

Un curso de HTML, CSS y JS te enseñará a crear páginas web usando lenguajes de marcado, estilos y programación, respectivamente.

Curso de Back-End

Un curso de Java, Spring, Hibernate y MySQL te enseñará a desarrollar aplicaciones web y de bases de datos utilizando el lenguaje de programación Java, el framework Spring para aplicaciones empresariales, el ORM Hibernate para la persistencia de datos y el sistema de gestión de bases de datos MySQL.

Curso de Figma

Un curso de Figma te enseñará a utilizar esta herramienta de diseño de interfaces de usuario para crear prototipos, diseños y animaciones para aplicaciones y sitios web.

Curso de bootstrap

Un curso de Bootstrap te enseñará a utilizar este framework de diseño web para crear sitios web y aplicaciones móviles responsivos y con un estilo visual moderno.

Curso de Back-End

Un curso de C/C++ te enseñará a programar en lenguaje C y C++, los cuales se utilizan ampliamente en el desarrollo de sistemas operativos, software embebido, videojuegos y aplicaciones de alto rendimiento, entre otros.

Curso de Python

Un curso de Python te enseñará a programar en uno de los lenguajes más populares y versátiles del mundo, utilizado en el desarrollo de aplicaciones web, análisis de datos, inteligencia artificial y muchas otras áreas de la informática.

Curso de Chat-GPT

Un curso de OpenAI te enseñará sobre la inteligencia artificial y cómo aplicarla para resolver problemas complejos en diversos campos, como el procesamiento del lenguaje natural, la visión por computadora y la robótica, utilizando las herramientas y tecnologías desarrolladas por OpenAI.
